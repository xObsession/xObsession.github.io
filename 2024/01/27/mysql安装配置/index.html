总结一下

1、进入mysql的安装目录下的bin目录（需要以管理员身份操作）

cd C:\mysql\mysql\bin

2、创建配置文件(可根据需要修改端口，默认3306)

my.ini

3、初始化数据库

mysqld –initialize

之后到data目录下找到.err文件 复制出密码

5.7和8.0版本需要初始化，以前版本提供data目录

4、开始安装（默认服务名是：mysql）

mysqld -install “服务名称”

mysqld -install mysql80

mysqld –remove mysql80

5、连接服务器

net start mysql80 启动服务

net stop mysql80 停止服务

6、登录

mysql -u root -p 初始密码 -P 端口号

7、修改密码

ALTER user ‘root‘@’localhost’ IDENTIFIED BY ‘123456’;
